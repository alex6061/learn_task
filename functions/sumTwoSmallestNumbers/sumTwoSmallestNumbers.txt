Создайте функцию, которая возвращает сумму двух 
наименьших положительных чисел для заданного массива минимум из 4 
положительных целых чисел. Не будут переданы числа с плавающей запятой или 
неположительные целые числа.

Например, когда массив передается как [19, 5, 42, 2, 77], вывод должен быть 7.


1. числа должны быть целыми
2. должны быть положительные

console.log(sumTwoSmallestNumbers([19, 5, 42, 2, 77])); // 7
console.log(sumTwoSmallestNumbers([15, 28, 4, 2, 43])); // 6
console.log(sumTwoSmallestNumbers([23, 71, 33, 82, 1])); // 24
